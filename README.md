# My Dotfiles

Currently just my neovim and bash configs, taken straight from my home directiory!
Note: my neovim config is heavily based on [typecraft's config](https://www.youtube.com/watch?v=zHTeCSVAFNY&list=PLsz00TDipIffreIaUNk64KxTIkQaGguqn), except slightly more minimal and with autoclose (and also 4 spaces for tabs instead of 2).

Also includes a basic pacman install script for packages I use for development.
This installs:
- git
- github-cli
- vim
- neovim
- konsole
- ttf-hack-nerd
- cmake
However it does not automatically apply my configurations.
